# todoList Django+Django restframework+react
-------
支持的功能：
* 增加一个待办事项
* 删除一个待办事项
* 标记一个待办事项为已完成
* 编辑一个待办事项的具体内容
* 列出所有的待办事项
* 待办事项可以设置优先级
* 支持按照不同的方式排序
